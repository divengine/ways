About this documentation
This book documents Div PHP Ways: its routing model, controller registration, hooks, rules, direct invocation, and CLI support. Ways treats application entry points as control points that can be composed on demand for web and command line applications.
  • Conceptual overview of ways and control points
  • Installation and basic usage
  • Closure and static method controllers
  • Hooks, rules, data flow, and arguments
  • HTTP and CLI routing examples
